### 1. Prerequisites

### 2. Environment Configuration

Update the following files with your actual credentials:

#### Backend Environment (`backend/.env`)
```env
# Update these values:
DISCORD_TOKEN=your_actual_bot_token
DISCORD_CLIENT_ID=your_actual_client_id
DISCORD_CLIENT_SECRET=your_actual_client_secret

# MongoDB Atlas connection string
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/discord-bot-dashboard








```

#### Frontend Environment (`frontend/.env`)
```env
# Update these values:
REACT_APP_DISCORD_CLIENT_ID=your_actual_client_id

```

### 3. Database Setup


#### MongoDB Atlas
1. Create a MongoDB Atlas account
2. Create a new cluster
3. Create a database user
4. Whitelist your server IP
5. Update the MONGODB_URI in backend/.env




### 4. Discord Bot Setup

1. Go to [Discord Developer Portal](https://discord.com/developers/applications)
2. Create a new application
3. Go to "Bot" section and create a bot
4. Copy the bot token to `DISCORD_TOKEN`
5. Go to "OAuth2" section
6. Copy Client ID and Secret to environment files
7. Set redirect URI to: `https://localhost/auth/discord/callback`

### 5. Deployment


#### Docker Deployment
```bash
# Build and start all services
docker-compose up -d --build

# Check status
docker-compose ps

# View logs
docker-compose logs -f
```
